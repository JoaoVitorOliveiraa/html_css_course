Pseudo-Elementos

Pseudo-elementos no CSS permitem estilizar partes específicas de um elemento, como a primeira letra ou inserir conteúdo antes/depois dele.

Sintaxe:

seletor::pseudo-elemento {
        propriedade: valor;
    }

Principais Pseudo-Elementos:

Conteúdo e Texto:

  • ::before → Insere conteúdo antes do elemento.
  • ::after → Insere conteúdo depois do elemento..
  • ::first-letter → Estiliza a primeira letra.
  • ::first-line → Estiliza a primeira linha.

Campos de Formulário:

  • ::placeholder → Estiliza o texto do placeholder de um input.
  • ::selection → Estiliza o texto selecionado pelo usuário.
