Pseudo-Classes

Pseudo-classes no CSS são palavras-chave que definem um estado especial de um elemento, permitindo estilização baseada em interações ou estrutura.

Sintaxe:

seletor:pseudo-classe {
        propriedade: valor;
    }

Principais Pseudo-Classes:

Interação:

  • :hover → Quando o usuário passa o mouse.
  • :focus → Quando um campo recebe foco.
  • :active → Quando um elemento é clicado.

Estados de formulário:

  • :checked → Quando um checkbox está marcado.
  • :disabled → Quando um campo está desativado.
  • :required → Quando um campo é obrigatório.

Estrutura:

  • :first-child → Primeiro filho de um elemento pai.
  • :last-child → Último filho de um elemento pai.
  • :nth-child(odd/even/n) → Seleciona filhos específicos.
