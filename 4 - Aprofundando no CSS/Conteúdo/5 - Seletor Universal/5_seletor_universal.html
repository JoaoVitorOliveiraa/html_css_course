Seletor Universal e Herança de Estilos

Seletor Universal

O seletor universal (*) no CSS seleciona todos os elementos da página, permitindo a aplicação de estilos globais.

Sintaxe:

* {
        propriedade: valor;
    }

Herança de Estilos

A herança de estilos no CSS é o mecanismo pelo qual certos estilos aplicados a um elemento pai são automaticamente transferidos para seus elementos filhos.
