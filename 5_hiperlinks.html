Hiperlinks

Tag A

Tag de âncora (anchor) que cria os hiperlinks.


Parâmetros/Atributos

Href

O link deve ser colocado dentro da string do parâmetro/atributo href.

Ex: href="link"

Quando não queremos colocar um link (link vazio), é uma boa prática escrever href="#".

Target

Para fazer com que o link seja aberto em outra aba, utiliza-se o parâmetro/atributo target.

Ex: target="blank" (abre o link em outra aba)

Ex: target="self" (abre o link na mesma aba)

OBS: O padrão da tag a sem o parâmetro target é de abrir na mesma aba.

ID

Para fazer com que o link te leve para algum lugar da sua página, é necessário colocar o id da tag de destino no parâmetro href.

Para adicionar um id a uma tag, basta colocar o parâmetro dentro da tag e dar um nome a ele.

Ex: tag id="nome-do-id"

OBS: No parâmetro href, tem que colocar uma hashtag antes do nome do id, pois é a forma de representar id's.

Ex: href="#nome-do-id"

Listas Não Ordenadas

Utilizada por meio da tag ul (unordered list).

Devemos colocar a tag de item de lista li (list item) dentro da ul, para representar cada item da lista.

Ex:

  • H1
  • H2
  • H3
  • H4
  • H5
  • H6
