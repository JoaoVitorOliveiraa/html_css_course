Input Pattern + Regex

O atributo pattern no input permite definir regras de validação usando expressões regulares (Regex). Isso garante que os usuários insiram dados no formato correto antes de enviar um formulário.

O atributo required em HTML é usado para tornar um campo obrigatório dentro de um formulário. Se um usuário tentar enviar o formulário sem preencher um campo marcado como required, o navegador impedirá o envio e exibirá uma mensagem de erro.

O atributo title no input é usado para fornecer uma dica ou mensagem adicional ao usuário. Ele é especialmente útil quando combinado com validações (required, pattern, maxlength, etc.), pois o navegador exibe essa mensagem caso o usuário insira um valor inválido.
