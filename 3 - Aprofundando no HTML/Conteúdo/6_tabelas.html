Tabelas

Tag table

A tag table cria tabelas para organizar dados em linhas e colunas. Principais tags/atributos: tr (linha), th (cabeçalho), td (célula de dado), caption (título da tabela), thead (cabeçalho), tbody (corpo), tfoot (rodapé), border (borda da tabela), colspan (mesclar colunas), rowspan (mesclar linhas).

Tag colgroup

A tag colgroup define um grupo de colunas dentro de uma tabela, permitindo a aplicação de estilos a colunas inteiras.
