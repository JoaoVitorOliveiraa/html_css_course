Tag q

Segundo ciclano:

A tag q em HTML é usada para definir citações curtas dentro de um texto. Diferente da blockquote, que é usada para citações longas, a q é destinada a pequenas citações e, por padrão, os navegadores adicionam aspas ao redor do conteúdo.
Assim como blockquote, a q também aceita o atributo cite para indicar a fonte da citação, embora o link não seja visível no navegador

Tags de Abreviações

Tag abbr

A tag abbr em HTML é usada para definir abreviações ou siglas, permitindo que os navegadores exibam informações extras quando o usuário passa o mouse sobre o termo. Isso melhora a acessibilidade e fornece contexto adicional.

Exemplo: O HTML é uma linguagem utilizada na criação de sites para a web.
