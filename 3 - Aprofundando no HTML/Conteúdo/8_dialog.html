Tag dialog

A tag dialog cria caixas de diálogo ou modais nativos no HTML, podendo ser abertas e fechadas via JavaScript. Principais atributos/métodos: open (exibe o diálogo), showModal() (abre como modal), show() (abre sem modal), close() (fecha o diálogo).
