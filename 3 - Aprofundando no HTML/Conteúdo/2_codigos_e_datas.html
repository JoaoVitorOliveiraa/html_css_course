Códigos e Datas

Tags de Código

Tag code

A tag code em HTML é usada para exibir trechos de código-fonte, mantendo uma fonte monoespaçada (como Courier) para melhor legibilidade. Ela é ideal para destacar comandos, variáveis, funções e pequenos trechos de código dentro de um texto.

Exemplo:

function () { console.log("Olá, mundo!") }

Tag pre

A tag pre em HTML é usada para exibir texto pré-formatado, preservando espaços, quebras de linha e indentação exatamente como estão no código-fonte.

Exemplo:

function () { 
        console.log("Olá, mundo!") 
    }

Tag var

A tag var em HTML é usada para representar variáveis em expressões matemáticas ou trechos de código. Por padrão, os navegadores exibem o conteúdo dentro de var em itálico, indicando que se trata de uma variável.

Exemplo:

var nome = 'João';

Dentro do código acima, a variável nome é igual a João.

Tag samp

A tag samp em HTML é usada para representar a saída de um programa ou sistema, como mensagens de erro ou respostas do terminal. Por padrão, os navegadores exibem o conteúdo dentro de samp em uma fonte monoespaçada, semelhante a code.

Exemplo:

Resposta de um computador.

Tag kbd

A tag kbd em HTML é usada para representar entradas do teclado, comandos digitados pelo usuário ou atalhos de teclado. Os navegadores, por padrão, exibem o conteúdo dentro de kbd em fonte monoespaçada para diferenciá-lo do restante do texto.

Exemplo:

Para copiar um texto, pressione Ctrl+C.

Tags de Data

Tag time

A tag time em HTML é usada para representar datas, horas ou períodos de tempo, permitindo que navegadores, motores de busca e assistentes digitais interpretem a informação corretamente. Ela pode incluir o atributo datetime para definir a data ou hora no formato ISO 8601.

Exemplo:

Dia 14 de Fevereiro de 2025
