Formulários

Tag form

A tag form em HTML é usada para criar formulários interativos que permitem ao usuário inserir e enviar dados para um servidor. Ela pode conter diversos elementos de entrada, como input, textarea, select, button, entre outros.

Principais atributos da form:

  • action: Define o destino do envio dos dados.
  • method: Define o método HTTP (GET ou POST).
  • enctype: Usado em uploads de arquivos (multipart/form-data).
  • autocomplete: Ativa/desativa preenchimento automático.
  • novalidate: Desativa a validação do formulário pelo navegador.

Tag label

A tag label em HTML é usada para definir rótulos (labels) para elementos de formulário, como caixas de texto, botões de rádio, checkboxes, entre outros. Ela ajuda a associar um texto descritivo a um campo de entrada, melhorando a acessibilidade e a usabilidade do formulário.

Tag input

A tag input em HTML é um dos elementos mais usados em formulários para receber dados dos usuários. Ela pode ser configurada com diferentes tipos, como texto, senha, e-mail, data, entre outros, através do atributo type. Além disso, o input pode ser combinado com outros atributos para personalizar o comportamento de cada campo.

Tag button

A tag button em HTML é usada para criar botões interativos. Eles podem ser usados para enviar formulários, disparar ações via JavaScript, ou até mesmo para interagir com o usuário de diversas formas. O button é muito mais flexível do que outras tags de botão, como o input type="submit", permitindo incluir HTML dentro dele, como texto, imagens ou ícones.
