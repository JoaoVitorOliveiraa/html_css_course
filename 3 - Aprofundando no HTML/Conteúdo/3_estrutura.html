Estrutura do HTML

Tag header

A tag header em HTML é usada para definir o cabeçalho de uma página ou de uma seção. Normalmente, contém elementos como títulos, logotipos, menus de navegação e introduções.

Tag nav

A tag nav em HTML é usada para definir uma seção de navegação da página, contendo links para outras páginas ou seções dentro da mesma página. Ela ajuda na organização do conteúdo e melhora a acessibilidade.

Tag main

A tag main em HTML é usada para definir o conteúdo principal de uma página, excluindo elementos como cabeçalhos (header), rodapés (footer) e barras laterais (aside). Ela melhora a acessibilidade e a estrutura do site.

Tag article

A tag article em HTML é usada para representar um conteúdo independente e autocontido, como posts de blog, notícias, comentários e artigos. Cada article pode ser lido separadamente sem perder o significado.

Tag aside

A tag aside em HTML é usada para representar conteúdo relacionado ao principal, mas que pode ser considerado complementar. É comum encontrá-la em barras laterais, caixas de informações extras ou links relacionados.

Tag section

A tag section em HTML é usada para agrupar conteúdos relacionados dentro de uma página. Cada section geralmente contém um título (h1 a h6) e pode incluir artigos, textos, imagens ou qualquer outro conteúdo relevante.

Tag div

A tag div em HTML é um elemento de bloco genérico usado para agrupar e estruturar conteúdos sem adicionar significado semântico. Normalmente, é combinada com CSS ou JavaScript para estilização e interatividade.

Tag footer

A tag footer em HTML é usada para definir o rodapé de uma página ou de uma seção. Normalmente, contém informações como direitos autorais, links de contato, políticas de privacidade e créditos.
