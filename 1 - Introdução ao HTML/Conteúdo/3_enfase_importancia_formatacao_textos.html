Ênfase, Importância e Formatação de Textos

Tag B/BOLD

Deixa o texto em negrito.

Tag STRONG

Define o texto como importante, além de deixá-lo em negrito.

Tag I

Deixa o texto em itálico.

Tag EM

Enfatiza o texto, além de deixá-lo em itálico.

Tag CODE

tag_code = "Tag para escrever códigos"

Tag BLOCKQUOTE

Este é um texto utilizando a tag blockquote.
Ela é geralmente usada para citações.

Tag PRE

Tag utilizada para representar texto pré-formatado. Muito utilizada para inserir códigos

Tag BR

Essa tag não necessita de fechamento.
Ela executa a função de quebra de linha.

Tag HR

Essa tag não necessita de fechamento, ela forma uma linha horizontal.

Tag SUP

Essa tag deixa o texto sobrescrito. Ex: x2

Tag SUB

Essa tag deixa o texto subscrito. Ex: fx(x,y)
