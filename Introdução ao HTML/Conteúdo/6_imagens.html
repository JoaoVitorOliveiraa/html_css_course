Imagens e seus Atributos

Tag IMG

Apresenta na tela do navegador uma imagem, carregada a partir de um arquivo externo ao código HTML.


Atributos:

  1. Src

    • O atributo src (source/fonte), representa de onde a imagem é retirada.
    • Pode ser de um arquivo local ou de um hospedado em outro lugar (sites).

  2. Alt

    • O atributo alt (alternative text/texto alternativo), é um texto para quando a imagem não carregar.

  3. Width

    • O atributo width determina a largura da imagem, normalmente expressada em pixels (Xpx).

  4. Height

    • O atributo height determina a altura da imagem, normalmente expressada em pixels (Xpx).

Tag FIGURE

Especifica conteúdo independente, como ilustrações, diagramas, fotos, listagens de códigos, etc.

Embora o conteúdo do elemento esteja relacionado ao fluxo principal, sua posição é independente do fluxo principal e, se removido, não deve afetar o fluxo do documento.


Atributos:

  1. Figcaption

    • O atributo figcaption é usado para adicionar uma legenda à imagem.
